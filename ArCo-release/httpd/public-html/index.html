Architettura della Conoscenza

Guida

Questo documento descrive brevemente le ontologie incluse nella versione v0.1 di ArCo, una rete di ontologie per descrivere i diversi tipi di beni culturali e le schede catalografiche ad esse associate, gestite con il sistema SIGEC dall’Istituto Centrale per il Catalogo e la Documentazione.

La versione v0.1 di ArCo è la prima ad essere pubblicata. Le ontologie che compongono la rete hanno tutte lo stato “alpha” che indica sia una condizione di instabilità (cioè le ontologie potranno subire modifiche anche sostanziali nelle prossime versioni) sia il fatto che le ontologie non sono state ancora sottoposte ad attività di collaudo sistematiche.

Esempi (SPARQL)

Quali eventi culturali hanno interessato il bene culturale X?

Quali sono le persone coinvolte nelle operazioni di redazione e aggiornamento della scheda sul bene culturale X, e con quali ruoli?

Quando è stata redatta e aggiornata la scheda sul bene culturale X?

PREFIX l0: <https://w3id.org/italia/onto/l0/>
PREFIX cis: <http://dati.beniculturali.it/cis/>
SELECT ?event ?eventName ?culturalProperty ?culturalPropertyLabel
WHERE{
  ?event cis:involves ?culturalProperty ;
    l0:name ?eventName .
  ?culturalProperty rdfs:label ?culturalPropertyLabel
}
PREFIX cataloguerecord: <http://www.w3id.org/arco/cataloguerecord/>
PREFIX roapit: <https://w3id.org/italia/onto/RO/>
SELECT *
WHERE{
  ?entity cataloguerecord:hasCatalogueRecordVersion ?record . 
  ?record cataloguerecord:hasCatalogueRecordRiT ?rit . 
  ?rit roapit:withRole ?role ;
    roapit:isRoleInTimeOf ?agent
}
limit 100
PREFIX cataloguerecord: <http://www.w3id.org/arco/cataloguerecord/>
SELECT *
WHERE{
  ?record cataloguerecord:hasCatalogueRecordVersion ?recordVersion . 
  ?recordVersion cataloguerecord:editedAtTime ?time .
  ?time rdfs:label ?timeLabel
}
limit 100

PREFIX arco: <http://www.w3id.org/arco/core/>
PREFIX locgeoamm: <http://www.w3id.org/arco/locgeoamm/>
SELECT *
WHERE{
  ?entity a arco:MovableCulturalProperty ;
    rdfs:label ?entityLabel ;
    locgeoamm:isInSite ?site .
  ?site rdfs:label ?siteLabel
}
limit 100
PREFIX arco: <http://www.w3id.org/arco/core/>

WHERE{

    rdfs:label ?entityLabel ;

  ?address rdfs:label ?addressLabel

limit 100

PREFIX cpdescription: <http://www.w3id.org/arco/cpdescription/>
SELECT ?entity ?entityLabel ?scope ?scopeLabel
WHERE{
  ?entity cpdescription:hasMibactScopeOfProtection ?scope ;
    rdfs:label ?entityLabel .
  ?scope rdfs:label ?scopeLabel
}
PREFIX identifier: <http://www.w3id.org/arco/identifier/>
SELECT ?entity ?id
WHERE{
  ?entity identifier:uniqueIdentifier ?id 
}
PREFIX identifier: <http://www.w3id.org/arco/identifier/>
PREFIX cataloguerecord: <http://www.w3id.org/arco/cataloguerecord/>
SELECT ?record ?catalogueRecordId ?entity
WHERE{
  ?record identifier:catalogueRecordIdentifier ?catalogueRecordId ;
    cataloguerecord:describes ?entity
}
LIMIT 100
