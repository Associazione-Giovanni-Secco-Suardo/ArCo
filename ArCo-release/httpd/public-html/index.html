Architettura della Conoscenza

Guida

Questo documento descrive brevemente le ontologie incluse nella versione v0.2 di ArCo, una rete di ontologie per descrivere i diversi tipi di beni culturali e le schede catalografiche ad esse associate, gestite con il sistema SIGEC dall’Istituto Centrale per il Catalogo e la Documentazione.

La versione v0.2 di ArCo è la seconda ad essere pubblicata. Fare riferimento a questo documento per la guida alla versione 0.1.

Esempi (SPARQL)

Quali eventi culturali hanno interessato il bene culturale X?

Quali sono le persone coinvolte nelle operazioni di redazione e aggiornamento della scheda sul bene culturale X, e con quali ruoli?

Quando è stata redatta e aggiornata la scheda sul bene culturale X?

PREFIX l0: <https://w3id.org/italia/onto/l0/>
PREFIX cis: <http://dati.beniculturali.it/cis/>
SELECT ?event ?eventName ?culturalProperty ?culturalPropertyLabel
WHERE{
  ?event cis:involves ?culturalProperty ;
    l0:name ?eventName .
  ?culturalProperty rdfs:label ?culturalPropertyLabel
}
PREFIX cataloguerecord: <http://www.w3id.org/arco/cataloguerecord/>
PREFIX roapit: <https://w3id.org/italia/onto/RO/>
SELECT *
WHERE{
  ?entity cataloguerecord:hasCatalogueRecordVersion ?record . 
  ?record cataloguerecord:hasCatalogueRecordRiT ?rit . 
  ?rit roapit:withRole ?role ;
    roapit:isRoleInTimeOf ?agent
}
limit 100
PREFIX cataloguerecord: <http://www.w3id.org/arco/cataloguerecord/>
SELECT *
WHERE{
  ?record cataloguerecord:hasCatalogueRecordVersion ?recordVersion . 
  ?recordVersion cataloguerecord:editedAtTime ?time .
  ?time rdfs:label ?timeLabel
}
limit 100

Quali sono tutte le localizzazioni che hanno riguardato il bene culturale X, di che tipo sono, e in quali intervalli temporali?

Quali sono i luoghi che hanno rivestito la funzione di deposito nei confronti di un certo bene culturale?

Quali sono i beni culturali immateriali descritti nelle schede attualmente pubblicate?

PREFIX arco: <http://www.w3id.org/arco/core/>
PREFIX locgeoamm: <http://www.w3id.org/arco/location/>
PREFIX tiapit: <http://www.w3id.org/italia/onto/TI/>
SELECT *
WHERE{
  ?entity a arco:CulturalProperty ;
    rdfs:label ?entityLabel ;
    locgeoamm:hasTimeIndexedQualifiedLocation ?location .
  ?location rdfs:label ?locationLabel ;
    locgeoamm:hasLocationType ?locationType ;
    tiapit:atTime ?locationTimeInterval .
}
limit 100

PREFIX arco: <http://www.w3id.org/arco/core/>
PREFIX locgeoamm: <http://www.w3id.org/arco/location/>
SELECT *
WHERE {
  ?entity a arco:CulturalProperty ;
    rdfs:label ?entityLabel ; 
    locgeoamm:hasTimeIndexedQualifiedLocation ?location ;
  ?location locgeomm:hasLocationType locgeoamm:StorageLocation .
}
limit 100

PREFIX arco: <http://www.w3id.org/arco/core/>
SELECT *
WHERE{
  ?entity a arco:IntangibleCulturalProperty
}

A quando risale la realizzazione del bene culturale X? E quale motivazione giustifica la cronologia?

Qual è lo stato di conservazione del bene culturale X? E quali interventi sul bene sono stati proposti?

PREFIX arco: <http://www.w3id.org/arco/core/>
PREFIX culturaldefinition <http://www.w3id.org/arco/culturaldefinition/>
SELECT ?entity, ?entityLabel, ?dating, ?datingSource, ?datingEvent
WHERE{
  ?entity a arco:CulturalProperty ;
    rdfs:label ?entityLabel ;
  culturaldefinition:hasDating ?dating . 
  ?dating culturaldefinition:hasSource ?datingSource ;
    culturaldefinition:hasEvent ?datingEvent .
  ?datingEvent l0:name ?eventName
  FILTER regex(?eventName, "realizzazione", "i")
}
limit 100
PREFIX arco: <http://www.w3id.org/arco/core/>
PREFIX cpdescription <http://www.w3id.org/arco/cpdescription/>
SELECT ?entity, ?entityLabel, ?conservationStatus, ?proposedIntervention
WHERE{
  ?entity a arco:CulturalProperty ;
    rdfs:label ?entityLabel ;
    cpdescription:hasProposedIntervention ?proposedIntervention ;
    cpdescription:hasConservationStatus ?conservation .
  ?conservationStatus cpdescription:hasConservationStatusType ?conservationStatus .	
}
